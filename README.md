# EzPlayTime
- Record the player's daily/weekly/monthly/total online time

## PlaceholderAPI
 - %ezplaytime_today_value%   / Daily online time（numbers）
 - %ezplaytime_today_display% / Daily online time（time format）
 - %ezplaytime_week_value%    / Weekly online time（numbers）
 - %ezplaytime_week_display%  / Weekly online time（time format）
 - %ezplaytime_month_value%   / Monthly online time（numbers）
 - %ezplaytime_month_display% / Monthly online time（time format）
 - %ezplaytime_total_value%   / Total online time（numbers）
 - %ezplaytime_total_display% / Total online time（time format）
 - %ezplaytime_last_online%   / Last online time
